## Prerequisites

* Java 11
* Maven 3.3+
* Docker 19.03+
* Docker Compose 1.25+

## Installation


```sh
./mvnw compile jib:dockerBuild
```


```sh
docker-compose up -d 
```

## Used Technologies

* Spring Boot 2.4.3
* Elasticsearch
* Kibana
* Spring Boot Web
* Content Negotiation Support(Xml,Json,Yaml Support)  
* Spring Boot Hateoas  
* Spring Boot Log4j2
* Problem Spring Web
* Spring Boot Actuator
* SpringDoc Openapi Web Mvc Core
* SpringDoc Openapi Web Ui
* Maven Jib Plugin
* Maven Clean Plugin
* Maven Enforcer Plugin
* Maven Compiler Plugin
* Lombok
* Mapstruct  
* Dev Tools
* Spring Boot Test
